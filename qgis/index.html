QGIS - Free and Open Source

QGIS is an open-source software project, an initiative under the Open Source Geospatial Foundation.

QGIS is free, stable, changes smoothly through time, with the source code available so that it can be extended as needed for specific tasks. It provides a graphical user interface and supports a wide variety of data types and formats.

QGIS provides a large number of basic GIS display and analysis functions. An interface has been developed with GRASS, another open-source GIS with complementary analytical functions.
